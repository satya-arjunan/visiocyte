Module visiocyte

Python module for interacting with the VISIOCYTE visualization program.

The visiocyte python module enables control of the VISIOCYTE biomedical volume visualization program from the python programming language. The visiocyte module exposes the VISIOCYTE plug-in API; thus most things that can be done from a VISIOCYTE plug-in can also be done interactively from the VISIOCYTE python console.
