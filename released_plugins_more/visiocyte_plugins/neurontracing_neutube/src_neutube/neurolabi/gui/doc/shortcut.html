Shortcut Keys

(Cmd refered to as Ctrl for PC)

General

Cmd+ZUndo
Cmd+Shift+ZRedo
Backspace/DeleteDelete selected objects
Cmd+ASelect all SWC nodes
CConnect selected SWC nodes
NConnect to the nearest SWC nodes (only work for a single selected node)
BBreak SWC node connections
ZLocate selected SWC nodes in 2D view

2D Window

Cmd + GTurn on adding SWC node
Cmd + RTurn on painting
Cmd + ETurn on erasing
[Shift] AMove image or swc node to left. Hold Shift for fast movement.
[Shift] DMove image or swc node to right
[Shift] WMove image or swc node up
[Shift] SMove image or swc node down
=/Up_ArrowZoom in
-/Down_ArrowZoom out
Left_ArrowDecrease Z position
Right_ArrowIncrease Z position
</Q Decrease radius
>/E Increase radius

3D Window

=Zoom in
-Zoom out
All ArrowsRotate
[Shift] ArrowsMove
< Decrease radius
> Increase radius
Cmd + SSave SWC
Cmd + PlusIncrease SWC Size Scale
Cmd + MinusDecrease SWC Size Scale
Cmd + MChange SWC Display Mode
