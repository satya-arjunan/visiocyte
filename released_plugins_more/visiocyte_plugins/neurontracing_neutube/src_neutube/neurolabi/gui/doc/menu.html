Menu

File

  • Open
    Open a file, which can be a tif or swc file.

  • Expand current
    • SWC
      Import one or multiple swc files into the current frame
    • Mask
      Import a mask file (png) into the current frame

View

  • Normal
    Show the current image stack in a normal way, i.e. one slice at a time.

  • Z Project
    Show the maximum intensity projection of the stack. Each channel is projected independently for a multi-channel stack.

  • 3D View
    Open the 3D view of the current frame.

  • 3D View w/o Volume
    Open the 3D view of the current frame without volume rendering of the image data. This is a fast way to show swc files in 3D.

  • Adjust > Brightness/Contrast
    Bring up the dialog of ajusting image brightness and contrast

  • Object
    • Dense
      Show objects (usually neuron reconstructions) in a dense way.
    • Sparse
      Show objects in a sparse way.
    • Skeleton
      Show the skeletons of objects only.

Tools

  • Process > Binarize
    Import one or multiple swc files into the current frame
  • Process > Mask
    Import a mask file (png) into the current frame

  • Trace
    • Mask -> SWC
      Convert the maks of the current frame into an SWC structure
